\tl{Bayesian inference}

Using the same data, use Bayes’ theorem to calculate the probability the percentage of yellow is 10\% and 20\%.

\[ P(10\% yellow | data) = \frac{P(data | 10\% yellow) \times P(10\% yellow)}{P(data)} \]

\vspace{7cm}

\[ P(20\% yellow | data) = \frac{P(data | 20\% yellow) \times P(20\% yellow)}{P(data)} \]
